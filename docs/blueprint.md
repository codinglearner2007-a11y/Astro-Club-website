# **App Name**: Cosmic Scroll

## Core Features:

- 3D Planet Rendering: Render interactive 3D planets with dynamic lighting using Three.js.
- Parallax Scrolling: Implement parallax scrolling using GSAP ScrollTrigger to control camera and object movement.
- Dynamic Text Illumination: Illuminate section text dynamically based on the proximity and light of nearby planets, powered by an LLM tool to calculate light and distance for readability.
- Animated Scene Transitions: Create smooth scene transitions between Hero, About, Events, Gallery, and Join Us sections.
- Gallery Integration: Showcase astronomy-related images and content within a 3D gallery environment.
- Club Event Display: Present club events on planets' surfaces in a chronological fashion. The newest event should be more prominent.
- Interactive Join Form: Include a 3D form that collects and sends user's emails when illuminated by a planet.

## Style Guidelines:

- Background color: Dark, desaturated blue-gray (#12182B) to evoke the void of space.
- Primary color: A vibrant, slightly desaturated electric purple (#9D6EEB) to add a striking, forward-looking element in this stellar context.
- Accent color: Analogous to the purple but shifted towards blue, (#6EA2EB), used for highlights and interactive elements.
- Headline font: 'Space Grotesk', sans-serif, for headlines and short descriptions.
- Body font: 'Inter', sans-serif, for longer text.
- Minimal, glowing icons representing each section and interactive element.
- Full-screen 3D environment with floating planets and text elements.
- Smooth, complex animations for planet movement, lighting effects, and scene transitions.
